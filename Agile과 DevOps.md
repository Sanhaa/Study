## DevOps
정의: 애플리케이션과 서비스를 빠른 속도로 제공할 수 있도록 조직의 역량을 향상시키는 문화 철학, 방식 및 도구의 조합
- 빠른 개발과 릴리즈
- 개발-운영 팀 간의 프로세스 자동화, 소통 중시

- Infrastructure as Code (인프라 구성, 테스트)
